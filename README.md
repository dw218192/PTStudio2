# PTStudio2

This is an ongoing effort to port PTStudio to use Nvidia's [Carbonite SDK](https://docs.omniverse.nvidia.com/kit/docs/carbonite/latest/index.html)

## Building
```
./repo${sh_ext} build
```
- replace ${sh_ext} with .sh on linux or .bat on windows

## Running
```
./repo${sh_ext} run
```
- replace ${sh_ext} with .sh on linux or .bat on windows
